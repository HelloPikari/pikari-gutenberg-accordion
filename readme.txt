Professional accordion/toggle blocks for WordPress Gutenberg editor with full accessibility support.

== Description ==

Create beautiful, accessible accordion content with this comprehensive WordPress Gutenberg block plugin. Perfect for FAQs, content organization, and improving user experience.

**Key Features:**

* **Full Accessibility Support** - Complete ARIA attributes, keyboard navigation, and semantic HTML
* **Multiple Icon Styles** - Choose from chevron, plus, arrow icons with left/right positioning
* **Smooth Animations** - Optional CSS transitions for expand/collapse effects
* **FAQ Schema Markup** - Built-in Schema.org FAQ structured data for SEO benefits
* **WordPress Integration** - Full support for core layout features (spacing, colors, borders)
* **Block Bindings Compatible** - Works seamlessly with Query Loop and dynamic content
* **Multi-expand Control** - Allow multiple items open or traditional accordion behavior

**Perfect for:**
* FAQ sections
* Product information
* Course content
* Documentation
* Any collapsible content needs

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pikari-gutenberg-accordion/`, or install through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In the block editor, add the "Accordion" block
4. Add "Accordion Item" blocks inside the Accordion container
5. Customize settings in the block inspector

== Frequently Asked Questions ==

= Is this plugin fully accessible? =

Yes! We follow WCAG guidelines with proper ARIA attributes, keyboard navigation support, and semantic HTML structure. The plugin is compatible with screen readers and other assistive technologies.

= Does it work with my theme? =

Yes! The plugin uses WordPress core layout features and follows coding standards for broad theme compatibility. It respects your theme's typography and color schemes.

= Can I use this with Query Loop blocks? =

Absolutely! The accordion blocks are fully compatible with WordPress Block Bindings API and work great with Query Loop for dynamic FAQ content.

= Does it support Schema markup? =

Yes! Enable FAQ schema markup in the accordion settings to add structured data for better search engine optimization.

= Can I customize the icons? =

Yes! Choose from multiple icon styles (chevron, plus, arrow) and position them on the left or right side of headings.
